Text Analysis of parliament speeches in German state parliaments

26 September, 2020

In my master thesis I analyzed the parliament speeches given in the German State parliaments between 2008 and 2018. The time period covered varies between the states. My python code that converts the speeches into a machine-readable format is publicly available. I’ve have created a dataverse where you can download the scraped speeches and interjections.
